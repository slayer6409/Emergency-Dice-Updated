Other Stuff

Custom Events

In the config there is an option to make custom events. Currently, you can make:

  • Enemy events: Spawns whatever quantity of selected enemy, either inside or outside.
  • Trap events: Spawns however many traps of whatever you select — you can make them move too.
  • Item events: Spawns whatever item you want with whatever quantity you want.

Brutal

There is a Brutal Company–style config that allows for dice effects to be triggered either randomly or at the beginning of the day.

April Fools

April Fools mode in the config makes dice float, increases spawn weights, and hides which dice you roll — for example, a Saint die might secretly be a Sacrificer.

Debug Menu

If you're the host of the lobby (and you have the debug menu/button enabled in the config), there's a robust debug menu for triggering dice effects, teleporting, reviving, spawning things, and more. The default keybind is Numpad -, but this can be changed in the settings.
